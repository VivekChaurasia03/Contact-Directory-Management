Contact Directory Management

Welcome to our Contact Directory Management application. This powerful tool helps you organize, store, and manage your contacts efficiently.

Key Features

  • Easy contact addition and editing
  • Advanced search capabilities
  • Contact categorization
  • Cloud-based image storage

Technologies Used

  • Spring Boot backend
  • Thymeleaf templating
  • Tailwind CSS for styling
  • Cloudinary for image storage

Get started by exploring the navigation menu to add, view, or search your contacts.
